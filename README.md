# Petit Cotton Website

A responsive e-commerce website for **Petit Cotton**, showcasing organic cotton products for babies and toddlers. The website includes a home page, product listings, shopping cart, and checkout functionality, built with modern web technologies.

---

## 🌐 Live Website

[View Live Website](http://womanly-sink.surge.sh/)

---

## 📝 Project Overview

Petit Cotton Website is designed to provide a seamless shopping experience. Key features include:

- **Responsive Navbar** with brand logo, navigation links, search bar, and cart badge.
- **Hero Section** with promotional content and call-to-action buttons.
- **Product Cards** showcasing products with images, colors, sizes, and prices.
- **Cart Page** with dynamic cart count and product listing.
- **Checkout Page** with shipping form, order summary, and payment options.
- **Newsletter Section** for email subscriptions.
- **Footer** with copyright and policy links.

---

## 💻 Technologies Used

- **HTML5** for structured content
- **Tailwind CSS** for styling and responsive design
- **DaisyUI** for ready-to-use UI components
- **JavaScript** for cart functionality and interactivity

## 🎨 Features

### Navbar

- Logo and brand name
- Navigation links: Shop, Collections, Sustainability, About
- Search bar
